# 产品需求文档

## 「一言为定」—— 桌面小组件 · 每日一句

**版本：** v1.0
**状态：** Draft
**作者：** Hermes Agent

---

## 一、产品概述

### 1.1 产品定位

一款**只有桌面小组件**的极简 Android App。用户安装后不需要打开主界面，直接在桌面添加小组件，每天显示一句精选的中文句子。

### 1.2 产品 slogan

> 每天早上，一句好话。

### 1.3 目标用户

- 喜欢桌面整洁、有仪式感的用户
- 古诗词/哲学/文学爱好者
- 希望每天被一句好话唤醒的人

---

## 二、核心功能

### 2.1 语料库

| 项目 | 说明 |
|------|------|
| 来源 | 内置离线 JSON，打包在 APK 中 |
| 总量 | ≥300 条，发布前充实到目标量 |
| 分类 | 古诗词（50%）/ 哲学（40%）/ 现代经典（10%） |
| 展示 | **用户端只显示句子 + 出处，不显示分类标签** |
| 更新 | 通过 App 版本更新迭代语料库，不设在线更新 |

### 2.2 桌面小组件

提供三种尺寸：

| 尺寸 | 展示内容 | 推荐场景 |
|------|---------|---------|
| 小（2×2） | 一句完整句子 + 出处 | 锁屏页、桌面边角点缀 |
| 中（4×2） | 句子 + 出处 + 当前日期/星期 | 桌面主屏，信息适中 |
| 大（4×4） | 句子 + 出处 + 日期 + 背景渐变 + 收藏按钮 | 桌面主角位 |

### 2.3 视觉规范

- **背景**：每日自动渐变色，根据日期 seed 生成，暖色/冷色交替变化
- **字体**：正文句子用衬线字体（优雅感），出处用无衬线小字
- **毛玻璃效果**：文字区半透明毛玻璃浮于背景之上
- **交互**：点击卡片 → 切换到下一条句子（翻页动画）
- **计数**：右上角显示「Day 1」「Day 2」... 记录使用天数

### 2.4 辅助功能

| 功能 | 说明 |
|------|------|
| 收藏 | 大号小组件含收藏按钮，收藏的句子存入本地 |
| 收藏夹 | App 内可查看/管理收藏的句子（这是唯一需要打开 App 的场景） |
| 复制 | 点击句子可复制到剪贴板 |
| 每日通知 | 可选：每天早 8 点推送一条当日句子（v1.0 暂不实现，v1.1） |
| 自定义导入 | 用户可通过文本文件导入自己的句子库（v1.0 暂不实现，v1.1） |

## 三、非功能需求

| 项目 | 要求 |
|------|------|
| 平台 | Android 8.0+（API 26+） |
| 安装包大小 | ≤ 5MB |
| 离线 | 完全离线运行，无需网络权限 |
| 权限申请 | 不需要任何敏感权限（无可疑权限） |
| 内存占用 | ≤ 50MB |
| 刷新机制 | 小组件每日自动刷新一次句子及背景样式（UTC 00:00） |

## 四、用户旅程

### 4.1 首次使用

```
下载安装 → 提示"添加到桌面小组件" → 用户退出到桌面
         → 长按桌面 → 选择「一言为定」
         → 选择尺寸 → 完成，看到第一句
```

### 4.2 日常使用

```
每天解锁手机 → 看到一句新句子 → 点击切换下一句
                                → 点击收藏喜欢句子
                                → 复制分享给朋友
```

### 4.3 收藏回顾

```
打开 App（唯一入口）→ 看到收藏列表 → 浏览/取消收藏
```

## 五、发布 & 分发

### 5.1 打包方式

- 使用 GitHub Actions CI/CD 自动打包
- 输出 Debug APK（可直接安装）
- 发布到 GitHub Releases 供下载

### 5.2 分发路径

```
GitHub 仓库 → GitHub Actions 打包 → Releases 页下载 APK
                                    → 通过网盘/本地传输到手机
```

> ⚠️ 微信/QQ 等即时通讯工具会拦截 .apk 文件，建议通过阿里云盘、奶牛快传、NAS 等方式传输。

## 八、风险 & 对策

| 风险 | 概率 | 影响 | 对策 |
|------|------|------|------|
| 小组件点击事件在部分 ROM 上不响应 | 中 | 高 | 使用 PendingIntent + BroadcastReceiver 方案，兼容主流 ROM |
| 渐变色在某些低端机上渲染卡顿 | 低 | 中 | 使用预计算色值，避免实时计算 |
| APK 被微信拦截无法传输 | 高 | 中 | 提前备好网盘方案，文档中注明替代传输方式 |
| 用户误以为需要联网权限 | 中 | 低 | 应用描述中明确标注"完全离线" |
